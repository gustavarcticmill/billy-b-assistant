**`trigger_session_start(source: str) -> None`** -- The core function. Per D-01:
- Declare globals: `is_active, session_thread, interrupt_event, session_instance, _last_trigger_time`
- D-04 global debounce: `now = time.time(); if now - _last_trigger_time < _DEBOUNCE_SECONDS: return; _last_trigger_time = now`

- Lock acquisition: `if not _session_start_lock.acquire(blocking=False):` -- copy the recovery logic from button.py lines 190-230 exactly (the stale lock recovery path), but replace references to the old module-level globals with the ones now in this module
- Session start block (inside `try:` after lock acquired):
  - D-05/WAKE-04: Notify hotword controller: `from .hotword import controller as _hw; _hw.notify_session_state(True); time.sleep(_MIC_HANDOFF_DELAY)`
  - Previous session cleanup (copy from button.py lines 234-269)
  - `audio.ensure_playback_worker_started(config.CHUNK_MS)`
  - `audio.playback_done_event.clear()`

    - `global session_instance, is_active`
    - `try: session_instance = BillySession(interrupt_event=interrupt_event); session_instance.last_activity[0] = time.time(); asyncio.run(session_instance.start())`
    - `except Exception as e: logger.error(f"Session error: {e}")`
    - `finally:` -- D-06/WAKE-09: `move_head("off"); is_active = False; session_instance = None;` then `_hw.notify_session_state(False)` (resume wake word immediately, controller's 2.0s cooldown prevents self-trigger); `logger.info("Waiting for trigger...", "Clock")`; `with contextlib.suppress(Exception): _session_start_lock.release()`

- `except Exception as e:` release lock
